# Offline SLM Agent

A fully offline AI agent built with Python, Ollama, and Qwen3:4B.

## Features

- Local LLM using Ollama
- Tool Calling
- ReAct Agent Loop
- Safe Calculator
- Note Management
- File Sandbox
- No internet required after model download

## Tech Stack

- Python
- Ollama
- Qwen3:4B
- python-dotenv

## Project Structure

```
src/
config.py
agent.py
tools.py
main.py
```
